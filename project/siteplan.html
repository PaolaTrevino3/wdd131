Glow Guide Site Plan

website planning document

Site Name

Glow Guide

I chose the name Glow Guide because the website will help visitors learn simple and practical skincare habits.

Optional domain:glowguidecare.com

Site Purpose

The purpose of this website is to provide beginner-friendly skincare information, including daily routines, product tips, skin type guidance, and basic care recomendations. The site will help users understand how to care for their skin without feeling overwhelm.

scenarios

  • what skin care routine should I follow if I'm a beginner
  • how do I know which products are best for my skin type?
  • what is the difference between morning and nighttime skincare?

Lora: used for headings and section titles to create an elegant and polished look.

Roboto: used for body text because it is clean, reliable and simple for online content.
